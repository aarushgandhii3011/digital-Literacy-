## 📖 About This Project

This repository is my Digital Literacy Portfolio, submitted as part of the **CSE0001 – Digital Literacy** course at VIT Bhopal University. The assignment scenario places me in the role of a **Student Digital Ambassador**, responsible for helping peers navigate the digital world safely and professionally.

The portfolio is organized across **five tasks**, each corresponding to one course module — covering infographic design, professional online profiles, coding platforms, email communication, and cybercrime awareness.
